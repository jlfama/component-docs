Lists

List styling can be applied to <md-list> elements by adding the pt-list class.

There are a few extra classes that can adjust the styling of lists.

Class Outcome
pt-list-dark Used when list is on a dark background so text and borders are tints of white.
pt-list-no-border Removes all borders on list items.
pt-list-no-top-border Removes top border from list.
pt-list-no-bottom-border Removes bottom border from list.
